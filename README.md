# Rijndael4plex
Rijndael library for BDASpecial-PlexPX

BDASpecial-PlexPXのビルドに必要な、aes.libを生成するためのソースコードです。
libObfuscateに含まれるRijndael.cppを基に改変されたものです。
このライブラリはLGPL 3.0ライセンスが適用されます。
